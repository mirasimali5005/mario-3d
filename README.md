## How to Run

1.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the Game:**
    ```bash
    python app.py
    ```

3.  **Play:**
    Open your browser and go to `http://127.0.0.1:5000`.

## Controls
- **WASD**: Move
- **Space**: Jump
